# DragVLA

Train your own VLA (Vision-Language-Action) model just using your mouse.

## 项目说明 / Project Description

DragVLA 是一个可视化的 Embodied AI 项目构建工具，通过拖拽和节点式界面，让你仅用鼠标即可搭建和配置 VLA 模型训练流程。

主要功能：

- **节点式流程构建**：基于 React Flow 的可视化节点编排，支持 VLM、Action Head、机器人控制、仿真、数据生成等节点
- **项目管理**：创建、打开和管理多个训练项目
- **技术栈**：React + TypeScript + Vite + Tailwind CSS + @xyflow/react

DragVLA is a visual project builder for Embodied AI. Build and configure your VLA model training pipeline through drag-and-drop and node-based UI.

## 快速开始 / Quick Start

```bash
# 安装依赖 / Install dependencies
npm install

# 启动开发服务器 / Start dev server
npm run dev

# 构建 / Build
npm run build
```
